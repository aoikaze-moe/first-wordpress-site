【日在风蓝032·办公室】一句话新闻

2014年 12月 29日 / 风蓝办公室 / 0 Comments

福岛机场命名为“奥特曼机场”（shenmegui）

此前日本就有消息报道称由于《名侦探柯南》作者青山刚昌老家就是鸟取县所以就将此地命名为“柯南机场”。

而近日位于日本福岛县的福岛机场宣布，正在考虑将机场的爱称定为“奥特曼机场”，有日本特摄之父称号的圆谷英二的出生地便是在福岛县的须贺川市，也是机场导入这个名称的初衷。

实际上福岛机场从2007年开始就积极的使用《奥特曼》系列来装点机场及举行各种活动，“奥特曼”已经成为福岛机场的代言人，对于使用“奥特曼机场”这个爱称，以前也有几次同样的议论，但是福岛县政府表现出了非常积极的姿态还是第一次。

另外对于使用这个名称还需要考虑到著作权等多方面的版权问题，距离正式决定还需要有非常多的时间。（喜羊羊机场，龙骧飞机场。。。） （更多…）

【日在风蓝024·办公室】一句话新闻（H）

2014年 12月 15日 / 风蓝办公室 / 0 Comments

剧场版第19弹《名侦探柯南 业火的向日葵》2015年4月18日首映

由青山刚昌创作的漫画《名侦探柯南》连载至今已有20个年头，期间推出的TV动画、OVA、剧场版等衍生作品均受到了广泛的关注。其中，剧场版可以说是粉丝们每年翘首以盼的重头戏了。自1997年的首部剧场版《名侦探柯南 引爆摩天楼》到今年的《名侦探柯南 异次元的狙击手》，总共献上了18部各有千秋的剧场版，而将于明年4月公映的第19弹剧场版的相关情报也终于在「周刊少年Sunday」2015年1号中得到解禁！

据悉，此次新作剧场版的标题为《名侦探柯南 业火的向日葵》（I火の向日葵）首映日定在了2015年4月18日。在原作者青山刚昌所绘的该作的海报上，我们可以看到奔跑于燃烧着的向日葵田中的柯南，与向下看着这一切的基德。然而，基德的脸上所浮现却是邪恶的笑容……海报上还标识了一句“你能解开吗？此艺术…”的宣传语。本作讲述了原本应在日本被烧毁的梵高的名画《向日葵》意外被发现，并且确定参与在纽约的拍卖。但在发现该名画的地方竟发生了杀人事件，而拍得这幅名画的铃木财团的飞机在运输该作品的途中也突发爆破，画也在那时被夺走。犯下这些罪行的真的是以这幅名画为目标的基德！？故事由此正式展开。
