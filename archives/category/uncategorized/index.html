2018年 11月 22日 / Tokamak / 0 Comments

首先告诉大家一个不幸的消息，官网大概要停更一段时间。

原因有很多啊，比如托卡我本人的学业考试等等啊，天气冷了大家也不愿意拍cos写稿子了啊，托卡我要被榨干了啊（？），总之就是有各种各样的原因。经过慎重的考虑，打算在这个时候停更了。

接手官网只有小半年，能写出这么多稿子也是我没有想到的。这里要感谢一下手绘组，歌团和cos组成员们的供稿，让我不至于没有东西可发。也要感谢小小，钦飏和诸葛，帮我联系组员们。当然，最后要感谢现在再看这个的你们。感谢你们愿意来看看这个官网，感谢你们没有嫌弃我每周一@（事实上我自己都觉得我太烦了）。当然，以后肯定还会再更新的啦，托卡什么时候咕过你们是吧？

嘛，总之，就是这么个情况了（摊手）。民那，我们明年再见！

【日在风蓝053·办公室】国漫《狐妖小红娘》即将推出单行本

2015年 11月 2日 / 风蓝办公室 / 0 Comments

      “万物有界，爱恨无由”。人气国漫，《狐妖小红娘》即将推出单行本！

      《狐妖小红娘》，由作者庹小新编绘，盘丝大仙上色。漫画本于《漫画show》杂志连载，后因《漫画show》停刊而转战腾讯动漫，每周双更，并成功于2015年6月26日动画化。而在最新一期漫画219话（总一百一十话）中，作者小新更是宣布了即将发行单行本的消息！这对于喜欢《狐妖》的读者们来说，不能不说是一个振奋的消息。而正在连载的动画也将进行到精彩阶段——女主苏苏即将苏醒，其真实身份是——？！（bilibili绝赞连载中！）想了解更多资讯，还请关注作者漫画哦O(∩_∩)O~
