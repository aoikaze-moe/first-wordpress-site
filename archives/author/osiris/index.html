【日在风蓝052·动漫放映】钢炼——叹息之丘的圣星

2015年 10月 29日 / 风蓝宣传部 / 0 Comments

风蓝娘将于10月31号（本周六晚）七点在东九A205举行人气动漫：钢之炼金师——叹息之丘的圣星 的放映活动！！！
《钢之炼金师》是由日本漫画家荒川弘创作的漫画作品，并于2003年正式动漫化。
故事发生在一个炼金术相当发达的世界，在这个世界的“炼金术”，是以理解物质的内在的法则：理解，分解，再构造成立，进行炼金术必须按照“等价交换”法则：为了获得某种东西，需要以同等的代价交换，代价不够的话便需要以自己的任何部份（身体的一部分、记忆等等）被作为代价的填补而被拿走。
爱德华·艾尔利克（Edward Elric）
本作主人公，个子矮小的少年，12岁取得国家炼金术师资格，拥有“钢”之称号。右手与左脚为机械义肢，为了见到去世的母亲，和弟弟阿尔冯斯一起进行了人体炼成，结果失去了弟弟的身体与自己的手脚，为了找回失去的一切，与弟弟阿尔冯斯踏上寻找贤者之石的旅途。结局为了唤回最重要的东西，而被留在门的另一边的世界。

阿尔冯斯·艾尔利克（Alphonse Elric）

本作的另一个主人公，爱德华的弟弟，原本是个金发金眼的人类少年，因在尝试复活母亲使用人体炼成时失败，结果整个身体都被夺走，爱德以右臂为代价换回了他的灵魂，只能让灵魂寄生在冰冷的钢铁盔甲里。为了找回失去的一切，与哥哥爱德华踏上寻找贤者之石的旅途。

温莉·洛克贝尔（Winry Rockbell）

本作女主角，失去了父母的她从小和祖母一起生活，与爱德和阿尔青梅竹马。对机械铠和跟机械铠有关的一切事物感兴趣，与祖母比拿可一样成为机械铠工程师，是爱德的专属技师。对艾尔利克兄弟来说是等同于家人的重要存在，为了能帮助一直在进行旅行的艾尔利克兄弟，努力提升自己，想让自己做出更好的机械铠，是个相当坚强而内心温柔的女孩。
不管你是想重温这部热血动漫，还是首次观看，10月31号（本周六）晚7点东九A205，《钢炼剧场版——叹息之丘的圣星》来袭！小伙伴们可以前去观看哟！

【日在风蓝051·文化节】玩转游园会

2015年 10月 20日 / 风蓝宣传部 / 0 Comments

由华中科技大学社团联合会举办的第十三届社团文化节将于10月23号（本周五）晚在西操拉开帷幕，文化节是由社联主办，华科各个社团参与，旨在表现社团风采的大型活动。

风蓝娘作为华科众多社团的一份子，文化节自然不会缺席，作为动漫社，风蓝娘每年游园会组织的节目都是相当有动漫社的风格，去年的看背影猜人物，捞金鱼以及今年即将在游园会组织的钩水球都是很有趣，很符合动漫迷的游戏哟！去游园会的同学一定要去玩哟，据说有神秘奖品呢！

作为一个要玩转文化节的社团，风蓝娘怎么会只准备游戏这么简单，不论是游园会当天的小舞台还是24号晚上的文化节开幕式风蓝娘都准备了精彩的节目，欢迎大家前去围观！！！

想了解更多关于风蓝娘的动态，请关注华中科技大学风蓝动漫社微博和微信公众平台。
